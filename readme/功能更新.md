## 2026-06-21 - 实现发射倒计时核心功能

### 新增
- 修改入口类 `KSPLaunchCountdownMod.cs`
  - 改为飞行场景加载（`KSPAddon.Startup.Flight`）
  - 初始化和协调所有子模块
- 新增预设语音包管理器 `PresetManager.cs`
  - 自动扫描 `Lauch Voice/` 目录下的子目录作为预设
  - 支持多语音包扩展（当前含 DFH-1）
- 新增音频播放器 `AudioPlayer.cs`
  - 使用WWW加载.ogg音频文件
  - AudioSource播放控制，支持播放结束回调
- 新增发射序列执行器 `LaunchSequence.cs`
  - SAS开启、满油门设置、分级操作
  - OnFlyByWire回调持续保持油门状态
- 新增倒计时控制器 `CountdownController.cs`
  - 协调完整发射序列：隐藏UI→开SAS→满油门→播放音频→分级→3秒后恢复UI
  - 支持取消倒计时操作
- 新增倒计时菜单UI `CountdownMenu.cs`
  - IMGUI窗口，预设选择列表
  - Launch/Cancel按钮
- 新增工具栏按钮管理 `ToolbarButton.cs`
  - ApplicationLauncher注册飞行场景按钮
  - 代码生成38x38临时图标（橙色圆形+白色"C"字符）
